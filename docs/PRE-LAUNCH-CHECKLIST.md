## ðŸ†˜ Quick Fix Guide

### If favicon doesn't show:
```
- Clear browser cache
- Hard refresh (Ctrl+Shift+R)
- Check file path is correct
- Wait 24 hours for browser cache
```

### If social preview doesn't show:
```
- Use Facebook Debugger to clear cache
- Verify OG tags are correct
- Check image URLs are absolute (not relative)
- Image should be at least 1200x630px
```

### If search doesn't work:
```
- Check search-page.js is loaded
- Verify article data is correct
- Check browser console for errors
```
